# web-musicplayer
音乐播放器，百度音乐没权限，只能播放自己服务器上的歌曲。。。
###[演示地址](http://hingsir.github.io/web-musicplayer/)

###Usage
```js
new MusicPlayer({
    container : document.getElementById('player'),
    songList:[
        'musics/lsjx.mp3',
        'musics/abj.mp3',
        'musics/10m.mp3',
    ],
    autoplay:false
})
```
